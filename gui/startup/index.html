Choose Safewallet mode
Be aware: Native mode requires to download the whole blockchain data to a local disk before you can start using it. This may take from several hours to a day depending on your connection and hardware. Please try to keep Safewallet running until the whole process is finished.

If you need a quick and easy access to your funds try Lite (SPV) mode which doesn't require any blockchain to be loaded locally. All data is requested on demand from Electrum servers. To gain access to Lite mode toggle "Enable experimental features" in Settings.
